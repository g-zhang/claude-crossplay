# NWL23 dictionary reference

Read this file in Dictionary mode or when strategic analysis depends on short
high-value words, vowel dumps, or bingo potential.

`dict.txt`, generated by `scripts/setup_dict.py`, is the authoritative word
list used by the solver. `scripts/nwl23_ref.py` contains smaller curated sets
and helpers for fast strategic lookups:

Dictionary setup fetches a pinned revision of the public `scrabblewords`
repository. Set `CROSSPLAY_WORDS_DIR` to a compatible git checkout, an
extracted source tree, or a flat directory containing the NWL23 text files to
reuse supplied files without a fresh download.

```python
from nwl23_ref import (
    TWO_LETTER,
    SHORT_J,
    SHORT_Q,
    SHORT_X,
    SHORT_Z,
    BINGO_STEMS,
    HI_PROB_7,
    VOWEL_DUMPS,
    I_DUMPS,
    U_DUMPS,
    is_valid_2letter,
    validate_crosswords,
    check_bingo_potential,
    get_premium_words,
    find_dump_words,
)
```

Use the resources as follows:

- Filter `dict.txt` for complete pattern, length, and rack-letter searches.
- Use `TWO_LETTER` for quick explanations of short hooks.
- Use `SHORT_J`, `SHORT_Q`, `SHORT_X`, and `SHORT_Z` when a rack contains a
  high-value tile.
- Use bingo stems and dump lists to explain rack-leave quality.
- The solver already validates generated main and cross words against
  `dict.txt`; do not reject a solver result merely because a word is absent
  from one of the curated subsets.
